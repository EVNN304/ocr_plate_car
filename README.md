# 🚗 Smart ALPR Pipeline

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)

[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Высокопроизводительный и отказоустойчивый конвейер (pipeline) для автоматического распознавания автомобильных номеров (ALPR/ANPR). Система использует двухэтапную детекцию на базе **YOLO**, распознавание текста через **PaddleOCR** и аппаратное ускорение **NVIDIA TensorRT**. 

Архитектура построена на мультипроцессинге с очередями, имеет встроенный Watchdog для автоматического восстановления после сбоев и поддерживает динамическую перезагрузку конфигурации (Hot-Reload) без остановки приложения.

## 🎯 Назначение и варианты использования

Проект представляет собой готовый промышленный бэкенд для компьютерного зрения. Идеально подходит для:
1. **СКУД и умные шлагбаумы**: Автоматическое открытие ворот по "белому списку".
2. **Платные парковки**: Бесконтактная фиксация номера для расчета стоимости.
3. **Мониторинг трафика**: Подсчет транспорта с фильтрацией по зоне интереса (ROI).
4. **Системы безопасности**: Мгновенное оповещение при обнаружении автомобилей из "черного списка" (структура JSON содержит поле `matched`).

---

## ✨ Ключевые особенности

* 🚀 **Максимальная производительность**: Автоматическая конвертация весов YOLO (`.pt`) в формат **TensorRT (`.engine`)** с поддержкой FP16, динамического батча и встроенного NMS.
* 🔄 **Истинный мультипроцессинг**: Разделение на независимые процессы (Чтение видео → Детекция → OCR) через `mp.Queue` для обхода GIL.
* 🛡️ **Watchdog (Отказоустойчивость)**: Фоновый поток мониторит дочерние процессы и автоматически перезапускает упавшие (с лимитом попыток).
* 🔥 **Hot-Reload конфигурации**: Изменение параметров (ROI, пороги `conf`, задержки) "на лету" через редактирование YAML без перезапуска.
* 📡 **Frontend-ready**: Встроенная кодировка кадров (Base64/JPEG) и формирование строго типизированных JSON-сообщений.

### 🔧 Пошаговая установка

#### **Шаг 1: Клонирование репозитория**

git clone https://github.com/EVNN304/ocr_plate_car.git

#### **Шаг 2: Создание виртуального окружения**
# Для Linux / macOS
python3 -m venv venv
source venv/bin/activate

# Для Windows
python -m venv venv
venv\Scripts\activate

#### **Шаг 3: Установка зависимостей**

pip install --upgrade pip
pip install -r requirements.txt

#### **Шаг 4: Настройка конфигурации**

Откройте файл config_pipeline2.yaml и проверьте параметры


#### **Шаг 5: Запуск пайплайна**
python __main__.py
